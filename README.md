FSW Store in Django

como iniciar o projeto

```bash
git clone https://github.com/patresio/fsw-store-py.git

cd fsw-store-py

python -m venv .venv

source .venv/bin/activate

pip install -r requirements.txt --no-cache-dir
```

Abrir 2 terminais
primeiro terminal execute

```bash
npm install

npm run dev
```

segundo terminal

```bash
python manage.py runserver
```
